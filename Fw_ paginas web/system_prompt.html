Navbar inteligente - Solidifica con blur al scrollear 80px - Marca el link activo según la sección visible - Cierra menú móvil al hacer clic en un link 4. Animaciones de entrada - Intersection Observer para fade-in al hacer scroll - Contador animado para las estadísticas del hero 5. Filtros de portafolio - Filtrado por categoría con animación de transición 6. Carrusel de testimonios - Auto-play cada 5 segundos, pausa en hover ## OPTIMIZACIÓN Y SEO - Meta tags completos (title, description, OG tags) - Semantic HTML (header, main, section, article, footer) - Alt text en todas las imágenes - Título de página: "[NOMBRE] — Desarrollador Web Django Python PHP | Freelance" - Meta description: "Desarrollador web freelance especializado en Django, Python, PHP..." - Schema.org markup para Persona/Freelancer - Colores con contraste WCAG AA ## TECNOLOGÍAS PERMITIDAS - Bootstrap 5 (CDN) para grid y componentes - Bootstrap Icons (CDN) para iconografía - Google Fonts para tipografía - Vanilla JavaScript únicamente (sin jQuery, sin frameworks) - CSS Custom Properties para theming ## ENTREGA Entrega UN SOLO ARCHIVO index.html con: - Todo el CSS en <style> dentro del <head> - Todo el JavaScript en <script> antes del </body> - Imágenes usando placeholder.co o unsplash para demos - Código limpio, comentado por secciones - Funcional al abrir directo en el navegador (sin servidor) Al terminar pregunta: "¿Quieres que ajuste colores, textos, secciones o agregue algo más?" # ================================================================ # FIN DEL SYSTEM PROMPT # ================================================================

Tips para atraer clientes con tu web
🔍

SEO Local

Agrega tu ciudad en el título: "Desarrollador Web en [Ciudad] — Django Python PHP". Google posiciona mejor resultados locales para búsquedas de servicios.

⚡

WhatsApp como CTA principal

Los clientes prefieren hablar por WhatsApp antes que llenar formularios. Pon el botón de WhatsApp visible en el hero, navbar y footer.

💰

Muestra rangos de precio

Los clientes serios quieren saber si puedes estar dentro de su presupuesto. Una sección de "Desde $X" filtra leads y ahorra tiempo de negociación.

🏆

Portafolio > Todo lo demás

Si no tienes proyectos reales, crea 2-3 proyectos de demo con código en GitHub. Un proyecto funcional convence más que 10 testimonios.

📊

Google Analytics

Agrega GA4 para saber cuántos visitan tu web y qué secciones leen. Pídele a Claude: "Agrega el código de Google Analytics G-XXXXXXXXX".

🚀

Deploy gratis

Sube tu index.html a GitHub Pages o Netlify. Es gratis, rápido y obtienes un dominio tuusuario.github.io en minutos.
